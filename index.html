que es LimeSurf

limesurf es un poyecto de un navegador web de codigo abierto escrito en python y con una interfaz pyqt5 usando el webview de qt5

el navegador tiene:

  • una interfaz facil de usar inspirado en navegadores modrenos como google chrome
  • soporte para extensiones y themes usando archivos .txt con codigo en python
  • optimisacion de memoria la app cuando un usuario no esta usando otras ventana las suspende para gastar menos recrusos
  • sistema de pestañas en cajas selladas cada pestaña se comporta como un proseso independiente si una se cierra las demas no
  • bloqueador de anuncios y convertidor de videos de youtube usando la api de yt-dlp
  • gestor de descargas liviano
  • soporte con estandares web actuales
  • y mas
